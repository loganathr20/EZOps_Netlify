Lightweight Automation Instructions

Lightweight Automation uses Github Triggers to Trigger the build and share logs.

Set trigger to "true" and email in https://github.com/loganathr20/EZOps_Netlify/blob/develop/Trigger_linuxhealth.txt.

Update and commit to Github

Build will trigger in the server as per cron schedule (usually 2 to 10 mins from the time of commit)

Build Logs will be shared thru email set in trigger file and in Webpage.
